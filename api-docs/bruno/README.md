# Huf Bruno Collection

This collection uses Bruno environment variables for sensitive values.

## Setup

1. Open Bruno and import this collection.
2. Create an environment (e.g., `local`).
3. Add the following variables:
   - `BASE_URL` — e.g., `http://localhost:8000`
   - `ADMIN_PASSWORD` — the Frappe administrator password
   - `API_KEY` — a sample AI provider API key for testing

Do not commit real credentials to this repository.
