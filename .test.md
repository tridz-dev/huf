## Changes Made

### 1. Fixed `frontend/src/hooks/dashboard/usePageData.ts`
**Before:**
- Complex ref tracking with `hasFetchedRef` and `fetchInProgressRef`
- Tried to detect `fetchFn` changes to re-trigger
- Inline `fetchFn` in `FlowListPage` caused new reference on every render
- Led to infinite loop of API requests (hundreds of `ERR_ABORTED`)

**After:**
- Simple `useEffect` with empty dependency array `[]` (runs once on mount)
- Proper cleanup with `cancelled` flag
- No dependency on `fetchFn` reference
- Stable loading state management

### 2. Frontend Build
- Rebuilt successfully with `yarn build`
