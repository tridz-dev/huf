# Welcome to Huf

Huf is a powerful Frappe application that enables you to create, manage, and integrate AI agents directly into your Frappe ecosystem. These agents can interact with your site's data, automate tasks, and provide intelligent assistance.

## What is Huf?

Huf allows you to:

- **Create AI Agents** - Build intelligent agents powered by leading AI models (OpenAI, Anthropic, Google, and 100+ more via LiteLLM)
- **Equip with Tools** - Give your agents the ability to interact with your Frappe site through CRUD operations, custom functions, and HTTP requests
- **Multiple Trigger Types** - Configure agents to run on schedules, document events, or through interactive chat interfaces
- **Complete Visibility** - Track every agent execution, conversation, and tool call with comprehensive logging

## Key Features

### 🤖 Flexible Agent Creation
Create agents with custom instructions, models, and parameters. Configure them for:
- **Event-Driven Automation** - Trigger agents based on DocType events
- **Scheduled Tasks** - Run agents at regular intervals
- **Interactive Chat** - Enable real-time conversations with agents

### 🛠️ Powerful Tool System
Equip agents with tools to interact with your Frappe site:
- **CRUD Operations** - Get, create, update, and delete documents
- **Custom Functions** - Create tools from your own Python functions
- **HTTP Requests** - Allow agents to interact with external APIs
- **Agent Chaining** - Enable agents to trigger other agents

### 📊 Comprehensive Logging
Track everything:
- **Agent Runs** - Every execution with status, tokens, and cost
- **Conversations** - Complete chat history
- **Tool Calls** - Detailed logs of every tool usage
